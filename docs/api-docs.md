# ACTRIS metadata catalog REST API

## Add metadata record

### URL

> POST https://dev-actris-md.nilu.no/metadata/add

### Parameters

| Parameter | Type | Required | Description
|:---------:|:----:|:--------:|:----------:|
| first_name | string | yes | - |
| last_name | string | yes | - |
